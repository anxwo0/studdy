Gym Workout 

# Пітч
Сайт для новачків та любителів спорту, який допомагає уникнути травм та підвищити ефективність тренувань завдяки демонстрації правильної техніки виконання вправ.

# MVP 
Базова версія сайту, яка повинна виконкувати ці функції:користувач може знайти потрібну вправу, переглянути відео з правильною технікою та прочитати опис залучених м'язів та типових помилок.

# Стейкхолдери
1. Основну цінність від сайту отримають новачки та любителі в спортзалі, які прагнуть покращити свої навички.
2. Оцінювати досягнення цілей MVP будуть перші користувачі через відгуки, а також девелопер під час перевірки проєкту.
3. На реалізацію та темп розробки впливатимуть мої власні ресурси та час як розробника.

# Мета першої ітерації
Запустити базову версію сайту, де найголовніші функції доступні та працюють коректно, щоб користувач міг успішно знайти та переглянути інструкцію до обраної вправи.

# Ознаки успіху (Definition of Success)
1. Відео з правильною технікою виконання відображується та відтворюється коректно.
2. Пошук вправи працює коректно і видає потрібний результат.
